# kannada-pos-aware-extractive-text-summariser
 “POS-aware extractive summarisation system for Kannada text using Stanza and Indic NLP.”

## Features
- Tokenization and POS tagging for Kannada
- Sentence scoring based on POS importance (NOUN, VERB, ADJ)
- Extractive summarisation (select top-k sentences)

## Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/Kruthi-Jin/kannada-pos-aware-extractive-text-summariser.git
cd kannada-pos-aware-extractive-text-summariser
